تحويل البرنامج إلى EXE على ويندوز

1) ثبّت Python 3 على ويندوز مع اختيار Add Python to PATH.
2) افتح المجلد.
3) شغّل الملف build_exe.bat كمسؤول أو تشغيل عادي.
4) انتظر حتى ينتهي البناء.
5) ستجد الملف النهائي هنا:
   dist\Package5_Warehouse.exe

ملاحظات:
- قاعدة البيانات ستُنشأ بجانب ملف التشغيل أو داخل نفس مجلد المشروع حسب طريقة التشغيل.
- للحفاظ على البيانات، ضع ملف EXE داخل مجلد ثابت وامنح المستخدم صلاحية الكتابة عليه.
- يمكن لاحقًا إضافة أيقونة رسمية للشركة عبر ملف .ico وتعديل أمر PyInstaller.
